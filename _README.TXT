o установить Java; oracle.com/in/java/
  установить R; r-project.org

o обратите внимание, что скрипту требуется установленный пакет XLConnect для чтения экселевских файлов
  для этого надо набрать в R следующую команду:
install.packages(c('XLConnect', 'rJava'), dependencies = c('depends','imports'))

o далее вводите список файлов в файл *-list.txt; в конце лишняя строка должна быть пустой
  прописать путь к Джаве в первой строке скрипта
  и вписываете актуальный путь во вторую строку скрипта
  
o далее выполняете скрипт - вписать в run.bat актуальный путь к R.exe и запустить через cmd
  он содержит в себе проверку данных на чистоту (повторы и опечатки, etc.)
    [например <U+0421> означает что там кириллическая С вписана]
    отчет можно посмотреть в файле *.Rout
  сначала их надо пофиксить - пока все тесты будут чистыми; потом прогнать финальный
